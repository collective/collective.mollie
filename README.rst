Introduction
============

collective.mollie provides a wrapper for the Mollie_ API. Currently
only the `iDeal API`_ is supported.

.. _Mollie: http://www.mollie.nl/
.. _`iDeal API`: http://www.mollie.nl/support/documentatie/betaaldiensten/ideal/
